# خطوات إضافة موظف جديد (Quick Start)

> دليل عملي سريع.

---

## الطريقة السريعة (لو عندك أدوار جاهزة)

### الخطوة 1: أضف الموظف
1. روح لـ **الموظفين** (`/persons/`).
2. اضغط **"إضافة موظف"**.
3. املى البيانات:
   - **البريد الإلكتروني**: ده اسم الدخول (مثلاً: `ahmed@test.com`).
   - **الاسم الأول + الاسم الرابع**.
   - **كلمة المرور**.
   - **الفرع الرئيسي**.
   - **فريق** (اختياري).
4. تأكد إن **"موظف"** و **"نشط"** مفعلين.
5. اضغط **"حفظ"**.

---

### الخطوة 2: اربطه بدور
1. روح لـ **أدوار الموظفين** (`/employee-roles/`).
2. اضغط **"إضافة دور موظف"**.
3. اختار:
   - **الموظف**: ahmed@test.com
   - **الدور**: مثلاً "موظف استقبال"
   - **الفرع**: الفرع اللي هيعمل فيه
4. اضغط **"حفظ"**.

---

### خلاص! الموظف جاهز.

جرب تسجيل الدخول بالبريد وكلمة المرور بتاعته.

---

## الطريقة البطيئة (لو مش عندك أدوار جاهزة)

لو لسه ما عملتش أدوار وصلاحيات قبل كده، اتبع الخطوات دي:

### أولاً: أنشئ الصلاحيات (Permissions)
روح لـ **الصلاحيات** (`/permissions/`) → اضغط "إضافة" → اعمل الصلاحيات دي:

| الكود | الاسم | التطبيق | الموديل | الإجراء |
|-------|-------|---------|---------|---------|
| `view_student` | عرض الطلاب | students | student | عرض |
| `add_student` | إضافة طالب | students | student | إضافة |
| `view_payment` | عرض المدفوعات | finance | payment | عرض |
| `add_payment` | إضافة مدفوعة | finance | payment | إضافة |
| `view_registration` | عرض التسجيلات | registrations | account | عرض |
| `add_registration` | إضافة تسجيل | registrations | account | إضافة |

(كرر لكل الإجراءات اللي محتاجها).

---

### ثانياً: أنشئ الدور (Role)
روح لـ **الأدوار** (`/roles/`) → اضغط "إضافة":

- **الدور**: "موظف استقبال"
- في قسم **الصلاحيات** (تحت) → اضغط "إضافة صلاحية للدور" → اختار الصلاحيات اللي عملتها.

---

### ثالثاً: ربط الموظف بالدور
روح لـ **أدوار الموظفين** (`/employee-roles/`) → اضغط "إضافة":

- الموظف + الدور + الفرع.

---

## أمثلة جاهزة للأدوار

| الدور | الصلاحيات |
|-------|-----------|
| **مدير نظام** | كل حاجة |
| **مدير فرع** | كل حاجة في فرعه |
| **موظف استقبال** | عرض+إضافة طلاب + تسجيلات + مدفوعات |
| **محاسب** | المدفوعات + سندات الصرف + الإيداعات |
| **مسؤول تسويق** | عروض الأسعار + عروض الطلاب + مكالمات |
| **مشرف دورات** | الدورات + التخصصات + التسجيلات |

---

## تلميح سريع

لو عايز تختصر كل ده:
1. اعمل **Superuser** (مدير نظام) → بيشوف كل حاجة.
2. اعمل **دور واحد** باسم "موظف" → حط فيه الصلاحيات الأساسية.
3. ادي الدور ده لكل الموظفين الجدد.
4. لو محتاج موظف له صلاحيات خاصة → اعملله دور منفصل.
